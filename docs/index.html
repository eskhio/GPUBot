#Get#

Windows .exe
npm npm i @eskh/gpubot -g && gpubot

#Use#

  1. Select which model you're interested in
  2. Provide your Discord login
  3. Wait for it

#Test#

  1. Join this test Discord channel
  2. Run the bot & select the test option when asked
  3. Send the message ping within #gpubot -> a GPU should be posted and the bot should open a Chrome window

#Build#

  1. git clone https://github.com/eskhio/GPUBot
  2. npm i --save-dev
  3. pkg . --target node17-win-x64 --out-path ./build/exe --compress GZip
